From Dirk
Changes to the Data Structure
  • No more log data - (specific clicks, page visits etc.)
  • No more comments data - (using Discuss to handle comments)
  • Badge data is a separate database - saving badge info, people's feedback
  • New courses (say in January) are stored in different tables now than the older courses 
    • for example, courses_course stores course info for newer courses since January
