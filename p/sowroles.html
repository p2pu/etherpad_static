Roles for School of Webcraft
 
 Alpha release (or later)

(note: we need to find better names)

  • Web Fundamentals Wiz
Description: A Web Fundamentals Wiz has a deep understanding of web fundamentals and consistently provides feedback and technical support to other members of the community.
Assessment: stealth (one get's this role by obtaining the required badges)
Rubric: 
  • has completed all the alpha challenges
  • shows exemplary web skills
  • provides others on a continuous basis with support around technical issues.
  • provides others with helpful feedback
Requirements: A web Fundamentals Wiz should have at least the following badges:
  • Webmaking 101 Basic Badge
  • HTML Basic Badge
  • DNS Basic Badge
  • Helpful Feedback Badge
 
  • Community Yogi
Description: A Community Yogi pushes work forward in the community by consistently providing feedback, sharing ideas and commenting and reviewing other peers work.
Assessment: stealth (one get's this role by obtaining the required badges)
Rubric: 
  • has completed all the alpha challenges
  • is an active member of the community
  • pushes work forward in the community
  • shares ideas threw blogs, the activity wall, and groups
  • provides others with helpful feedback
Requirements: A Community Yogi should have at least the following badges:
  • Webmaking 101 Basic Badge
  • Super Blogger Badge 
  • Helpful Feedback Badge
  • Team Player Badge
  • Social Legend Badge
 
  • Webcraft Ninja
Description: A Webcraft Ninja has a deep understanding of web fundamentals,consistently shares work, provides feedback and technical support to other peers and is a leader or a mentor in that sense to the community.
Assessment: stealth (one get's this role by obtaining the required badges)
Rubric: 
  • has completed all the alpha challenges
  • shows exemplary web skills
  • provides others on a continuous basis with support around technical issues.
  • provides others with helpful feedback
  • is an active member of the community
  • pushes work forward in the community
  • shares ideas threw blogs, the activity wall, and groups
Requirements: A Community Yogi should have at least the following badges:
  • Webmaking 101 Basic Badge
  • HTML Basic Badge
  • DNS Basic Badge
  • Super Blogger Badge 
  • Helpful Feedback Badge
  • Team Player Badge
  • Social Legend Badge

  • Tourist (negative role)
Description: A  Tourist is an inactive member of the community who has started but not completed the challenges. The webcraft tourist provides no feedback to others, is inconsistent with sharing work and reviewing other's work.
Requirements: 
  • Has been inactive for x period of time
  • Has started not completed tasks
  • Has no badge (optional)
Rubric:
  • has completed none or few of the alpha challenges
  • is an inactive member of the community
  • rarely or never shares ideas threw blogs, the activity wall, and groups or provides others with feedback
